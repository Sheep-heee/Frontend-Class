1) 전체선택자
 - wildcard 선택자
 - reset css에서 씀
 - * (전체를 의미하는 선택자)

2) 태그선택자
 - 특정한 태그를 선택해서 스타일을 적용

3) 아이디선택자
 - html 태그 중에서 id 속성값을 가지고 있는 태그들에게 스타일을 적용
 - "#"가 맨 앞에 붙어야 함
 - id 속성값은 반드시 html에서 1번만 사용

4) 클래스선택자
 - html 태그 중에서 class 속성값을 가지고 있는 태그들에게 스타일을 적용
 - "."이 맨 앞에 붙어야 함
 - class 속성값은 html에서 여러번 횟수 제한 없이 사용 가능
 - 반복적으로 공통된 스타일 정의하고 적용

5) 후손선택자
 - 부모 아래에 들어오는 모든 자식 요소들
 - 자손선택자도 포함
 - " " 띄어쓰기

6) 자손선택자
 - 부모의 직계 자식 요소
 - ">" 자손선택자 기호

7) 속성선택자
 - 동일한 html 태그의 동일한 속성을 가지고 있는 경우, 안에 들어가 있는 속성값에 따라서 스타일을 상이하게 적용해야 할 때 사용함.
 - 태그 선택자 뒤에 대괄호 사용 [ ]

8) 동위선택자
 - 같은 위치에 속해 있는 요소들 간 스타일을 적용 -> 반드시 형제 요소여야 함
 - 특정 요소의 바로 뒤에 존재하는 형제 요소 찾을 때는 "+"
 - 특정 요소 뒤에 존재하는 모든 형제 요소 찾을 때는 "~"

9) 구조선택자

10) 반응선택자
 - 사용자가 어떤 액션(*이벤트) 실행 -> 반응 일으킴
 - 마우스 오버(:hover), 클릭(:active)

11) 상태선택자
 - 사용자의 이벤트 액션을 통해서 특정 태그 및 요소들의 상태가 변화되었다는 것을 감지하고 이를 통해서 스타일 적용
 - form : 버튼 / check / radio
 - 선행선택자 뒤에 :를 붙임
